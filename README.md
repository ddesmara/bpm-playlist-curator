# BPM Playlist Curator

## Overview
An AI-assisted decision-support tool that helps fitness instructors curate workout playlists based on BPM and energy flow across workout phases.

## Data Source
- Spotify Audio Features API
- Public, non-user-specific metadata

## Project Structure
- notebooks/: Data ingestion, analytics, and recommendation logic
- data/: Data schemas and documentation (no raw data committed)

## Key Features
- BPM-based workout phase assignment
- Transition smoothness optimization
- Explainable clustering logic

## Tech Stack
- Python
- Pandas, NumPy, scikit-learn
- Spotify API
- Google Colab

## Status
In progress
